    - Bidirectional Path Tracing
    - Photon Mapping
    - AOV Rendering

    - Lambertian Diffuse
    - Specular Reflection
    - Specular Transmission
    - Specular Dielectric
    - Modified Phong
    - Mixed

    - Sphere
    - Triangle
    - Rectangle
- ### Light
    - Point Light
